# CC1101 – Pinout (SPI)

This document defines the pin mapping between the **CC1101**
and an **Arduino Uno / Arduino Nano**.

## CC1101 ↔ Arduino Uno / Nano

| CC1101 Pin | Arduino Pin |
|-----------|-------------|
| GND       | GND |
| VCC       | 3.3V |
| CSN       | D10 |
| SCK       | D13 |
| MOSI      | D11 |
| MISO      | D12 |
| GDO0 / GDO2 | D3 |
